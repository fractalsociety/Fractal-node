Address lookup

Paste a 20-byte account / contract address. Balance, nonce, and bytecode length use eth_getBalance, eth_getTransactionCount, and eth_getCode. You can omit the 0x prefix; use a transaction lookup below, then click From / To to jump here.

Transaction lookup

eth_getTransactionByHash + receipt. After lookup, use the address buttons to inspect an account.

About transaction hashes

Block lists show whatever hash the RPC returns — often keccak256(EIP-1559 raw) for MetaMask txs, sometimes keccak256(borsh) for native-only paths. Use the same JSON-RPC endpoint you submitted on (usually the producer). Details: docs/explorer.md in this repo.
